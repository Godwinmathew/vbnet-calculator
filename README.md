# VB.NET Calculator

A simple desktop calculator built using **VB.NET Windows Forms in Visual Studio**.

## 📌 Features
- Addition
- Subtraction
- Multiplication
- Division
- Decimal number support
- Simple GUI using Windows Forms

## 🛠 Technologies Used
- VB.NET
- Windows Forms
- Visual Studio

## 📂 Project Structure

vbnet-calculator
│
├── CalculatorVB.slnx
├── CalculatorVB
│   ├── Form1.vb
│   ├── Form1.Designer.vb
│   ├── Form1.resx
│   ├── CalculatorVB.vbproj
│   └── ApplicationEvents.vb
│
└── README.md


## ▶ How to Run
1. Clone the repository
2. Open `CalculatorVB.slnx` in Visual Studio
3. Click **Run (▶)** to start the application
